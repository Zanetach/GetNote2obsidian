# Changelog

## 1.0.1 - 2026-04-13

### Fixes
- 修复无限设备授权在已安装插件中被误判为无效的问题
- 重新构建并同步发布产物，确保运行中的插件与源码逻辑一致

## 1.0.0 - 2026-04-13

### Features
- 发布 GetNote Sync 1.0.0 正式版本
- 支持 Get笔记录音转写同步到 Obsidian
- 支持发芽报告、主题索引页、原始逐字稿与会议纪要结构
- 支持基于设备机器码的 License 授权
- 支持单 license 多设备授权，默认最多 3 台，可选无限设备
- 提供命令行、交互式与 macOS 双击版发码工具
